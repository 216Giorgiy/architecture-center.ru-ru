## <a name="synchronous-versus-asynchronous-messaging"></a><span data-ttu-id="168fb-146">Синхронный и асинхронный обмен сообщениями</span><span class="sxs-lookup"><span data-stu-id="168fb-146">Synchronous versus asynchronous messaging</span></span>

<span data-ttu-id="168fb-147">Существует два основных шаблона обмена сообщениями, с помощью которых микрослужбы могут взаимодействовать.</span><span class="sxs-lookup"><span data-stu-id="168fb-147">There are two basic messaging patterns that microservices can use to communicate with other microservices.</span></span>
