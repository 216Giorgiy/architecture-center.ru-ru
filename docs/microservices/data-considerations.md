## <a name="challenges"></a><span data-ttu-id="476a0-123">Сложности</span><span class="sxs-lookup"><span data-stu-id="476a0-123">Challenges</span></span>

## <a name="approaches-to-managing-data"></a><span data-ttu-id="476a0-134">Подходы к управлению данными</span><span class="sxs-lookup"><span data-stu-id="476a0-134">Approaches to managing data</span></span>
