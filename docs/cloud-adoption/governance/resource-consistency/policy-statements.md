## <a name="ungoverned-subscriptions"></a><span data-ttu-id="f0bae-122">Неуправляемые подписки</span><span class="sxs-lookup"><span data-stu-id="f0bae-122">Ungoverned subscriptions</span></span>

<span data-ttu-id="f0bae-123">**Технический риск**.
