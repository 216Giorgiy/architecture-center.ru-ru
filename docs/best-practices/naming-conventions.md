## <a name="naming-rules-and-restrictions"></a><span data-ttu-id="13b20-181">Правила именования и ограничения</span><span class="sxs-lookup"><span data-stu-id="13b20-181">Naming rules and restrictions</span></span>

<span data-ttu-id="13b20-182">Каждый тип ресурса или службы в Azure связан с набором ограничений и областей применения имен.
