## <a name="naming-rules-and-restrictions"></a><span data-ttu-id="33228-181">Правила именования и ограничения</span><span class="sxs-lookup"><span data-stu-id="33228-181">Naming rules and restrictions</span></span>

<span data-ttu-id="33228-182">Каждый тип ресурса или службы в Azure связан с набором ограничений и областей применения имен.
