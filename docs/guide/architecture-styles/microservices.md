- <span data-ttu-id="b827c-191">**Набор навыков.**</span><span class="sxs-lookup"><span data-stu-id="b827c-191">**Skillset**.</span></span> <span data-ttu-id="b827c-192">Микрослужбы являются распределенными системами.</span><span class="sxs-lookup"><span data-stu-id="b827c-192">Microservices are highly distributed systems.</span></span> <span data-ttu-id="b827c-193">Тщательно оцените знания и опыт своей команды для работы с этими службами.</span><span class="sxs-lookup"><span data-stu-id="b827c-193">Carefully evaluate whether the team has the skills and experience to be successful.</span></span>
