<span data-ttu-id="bab7c-103">В решениях для работы с большими данными часто используются длительные пакетные задания для фильтрации, статической обработки и других процессов подготовки данных для анализа.</span><span class="sxs-lookup"><span data-stu-id="bab7c-103">Big data solutions often use long-running batch jobs to filter, aggregate, and otherwise prepare the data for analysis.</span></span> <span data-ttu-id="bab7c-104">Обычно в эти задания входит чтение исходных файлов из масштабируемого хранилища (например HDFS, Azure Data Lake Store и служба хранилища Azure), их обработка и запись выходных данных в новые файлы в масштабируемом хранилище.</span><span class="sxs-lookup"><span data-stu-id="bab7c-104">Usually these jobs involve reading source files from scalable storage (like HDFS, Azure Data Lake Store, and Azure Storage), processing them, and writing the output to new files in scalable storage.</span></span>
