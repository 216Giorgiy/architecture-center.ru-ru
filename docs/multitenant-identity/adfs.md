<span data-ttu-id="c4161-108">Одним из вариантов для таких пользователей является синхронизация локальной службы AD с Azure AD с помощью [Azure AD Connect].</span><span class="sxs-lookup"><span data-stu-id="c4161-108">One option is for these customers to sync their on-premise AD with Azure AD, using [Azure AD Connect].</span></span> <span data-ttu-id="c4161-109">Но, возможно, некоторым клиентам не удастся реализовать этот подход ввиду особенностей корпоративной ИТ-политики или по другим причинам.</span><span class="sxs-lookup"><span data-stu-id="c4161-109">However, some customers may be unable to use this approach, due to corporate IT policy or other reasons.</span></span> <span data-ttu-id="c4161-110">В этом случае следует рассмотреть другой вариант — создание федерации с помощью служб федерации Active Directory (AD FS).</span><span class="sxs-lookup"><span data-stu-id="c4161-110">In that case, another option is to federate through Active Directory Federation Services (AD FS).</span></span>
